## Setup

### Prerequisites

- R version 4.0 or higher
- RStudio (recommended)
- Git
- GitHub account

### Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:KJangTucci/R-Project.git
   cd R-Project
   ```

2. Install required packages:
   ```R
   install.packages(c("devtools", "testthat", "roxygen2"))
   devtools::install()
   ```
